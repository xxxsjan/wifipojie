## 本地安装方法：

- 下载解压好以后，我们用 cmd 命令行,进入到你的文件目录，
  cd pywifi-master

- 使用命令 pip install . 注意了（install 后面有个点）
  pip install .
- 然后就会安装了，等一会就可以了。

- 执行脚本

python py_wifi.py
